# Support

Start with [Getting started](docs/getting-started.md) and [Troubleshooting](docs/troubleshooting.md). For a source install, run:

```bash
uv run --no-dev --frozen handoff-forge doctor
```

If you still need help, use the repository's bug or feature-request form. Include:

- Handoff Forge version and install method;
- operating system and Python version when using a source install;
- minimal steps using synthetic data;
- sanitized diagnostic output and the exact error category.

Never attach provider keys, private source files, generated private handoffs, the application-data directory, or unreviewed logs. Replace local usernames and paths before posting.

Use GitHub private vulnerability reporting for security or privacy vulnerabilities. Do not open a public issue containing exploitable samples, credentials, or confidential data.
